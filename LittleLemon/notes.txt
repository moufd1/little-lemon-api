## Endpoints API

### Authentification
- POST /auth/users/ - Créer un utilisateur
- POST /auth/token/login/ - Obtenir un token
- POST /auth/token/logout/ - Se déconnecter

### Menu Items
- GET /api/menu-items/ - Lister les articles (tous les utilisateurs)
- POST /api/menu-items/ - Créer un article (managers seulement)
- GET /api/menu-items/{id}/ - Détail d'un article
- PUT/PATCH /api/menu-items/{id}/ - Modifier un article (managers)
- DELETE /api/menu-items/{id}/ - Supprimer un article (managers)

### Categories
- GET /api/categories/ - Lister les catégories
- POST /api/categories/ - Créer une catégorie (managers)

### User Groups
- GET/POST /api/groups/manager/users/ - Gérer les managers
- DELETE /api/groups/manager/users/{id}/ - Retirer un manager
- GET/POST /api/groups/delivery-crew/users/ - Gérer l'équipe de livraison
- DELETE /api/groups/delivery-crew/users/{id}/ - Retirer de l'équipe

### Cart
- GET /api/cart/menu-items/ - Voir le panier
- POST /api/cart/menu-items/ - Ajouter au panier
- DELETE /api/cart/menu-items/ - Vider le panier

### Orders
- GET /api/orders/ - Lister les commandes
- POST /api/orders/ - Créer une commande
- GET /api/orders/{id}/ - Détail d'une commande
- PUT/PATCH /api/orders/{id}/ - Modifier une commande
- DELETE /api/orders/{id}/ - Supprimer une commande (managers)
