## API Endpoints

### Authentication
- POST /auth/users/ - Register new user
- POST /auth/token/login/ - Get authentication token
- POST /auth/token/logout/ - Logout

### Menu Items
- GET /api/menu-items - List all menu items (public)
- POST /api/menu-items - Add menu item (Manager only)
- GET /api/menu-items/{id} - Get single menu item (public)
- PUT/PATCH /api/menu-items/{id} - Update menu item (Manager only)
- DELETE /api/menu-items/{id} - Delete menu item (Manager only)

### Categories
- GET /api/categories - List all categories (public)
- POST /api/categories - Add category (Manager only)

### User Group Management
- GET /api/groups/manager/users - List managers (Admin only)
- POST /api/groups/manager/users - Add user to manager group (Admin only)
- DELETE /api/groups/manager/users - Remove user from manager group (Admin only)
- GET /api/groups/delivery-crew/users - List delivery crew (Manager+ only)
- POST /api/groups/delivery-crew/users - Add user to delivery crew (Manager+ only)
- DELETE /api/groups/delivery-crew/users - Remove user from delivery crew (Manager+ only)

### Cart Management
- GET /api/cart/menu-items - View current user's cart (Authenticated)
- POST /api/cart/menu-items - Add item to cart (Authenticated)
- DELETE /api/cart/menu-items - Clear cart (Authenticated)

### Orders
- GET /api/orders - List orders (role-based filtering)
- POST /api/orders - Create order from cart (Customer)
- GET /api/orders/{id} - Get specific order (role-based access)
- PUT/PATCH /api/orders/{id} - Update order (Manager/Delivery crew)
- DELETE /api/orders/{id} - Delete order (Manager only)

## How to Test
1. Start server: `pipenv run python manage.py runserver`
2. Access Django admin: http://127.0.0.1:8000/admin/
3. Test API endpoints: http://127.0.0.1:8000/api/
4. Use Insomnia/Postman for API testing
